Delta-Neutral Trading System

Option Dash is a high-performance, real-time data visualization dashboard designed for options traders. It provides a live view of critical market indicators, strike zones, and volatility trends to enable data-driven decisions.

Project Overview

Option Dash empowers traders with a zero-latency interface for monitoring fast-moving markets. Built on a scalable architecture using Kafka for high-frequency data ingestion and FastAPI for real-time delivery, the system processes complex options Greeks and market zones on the fly. It organizes intraday data into persistent JSON storage for efficient retrieval and historical analysis.

Workflow & Architecture

1

Data Ingestion

A dedicated background consumer connects to a Kafka broker to ingest live "option-data" messages at high frequency.

2

Processing Pipeline

Incoming messages are parsed in real-time. The system calculates key risk zones (Green/Amber/Red) and updates Greeks like Delta and Theta.

3

Smart Storage

Processed data is cached and written to date-stamped JSON files organized by account_id, ensuring fast retrieval without stressing the main database.

4

Real-Time API

FastAPI endpoints serve the latest processed financial data to the dashboard with minimal latency.

5

Dynamic Visualization

The frontend polls the API every second, instantly updating the Price Indicator Bar and IV trend charts to reflect current market state.

Key Features

📊

Live Indicator Bar

A unique visual representation of Spot Price relative to Call/Put Strikes and risk zones (Green/Amber/Red).

📈

IV Trend Tracking

Real-time line charts utilizing Chart.js to track Implied Volatility (IV) spikes for both Call and Put options.

👥

Multi-Account Support

Seamlessly toggle between different trading accounts via a dynamic dropdown to monitor multiple portfolios.

⚡

Zero-Latency Feel

Optimized frontend polling and backend delivery ensure the UI feels alive and synchronized with the market tape.

📂

Historical Snapshots

Automated daily file rotation ensures that past trading sessions are organized and available for review.

⏰

Timezone Optimized

Fully optimized for IST (Indian Standard Time) to align perfectly with NSE/BSE market trading hours.
