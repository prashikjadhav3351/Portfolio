Edtube — AI Learning Platform

An intelligent AI-powered learning and testing ecosystem designed to personalize student education. Using LLMs and structured data processing, Edtube generates concept-based tests, evaluates logical understanding, and provides actionable performance insights.

Project Overview

Edtube is a comprehensive platform where students can register, select their class levels, and engage in structured learning through AI-generated tests. The system processes academic material into structured JSON data, allowing for precise question generation categorized by chapters and interconnected concepts. By leveraging Bloom’s Taxonomy, the platform dynamically adjusts test difficulty and provides in-depth feedback on conceptual gaps, mistakes, and areas for improvement.

System Workflow

1

Student Registration

Students register with a unique ID and select their class level (e.g., 7th, 8th). This profile customizes their learning path and available content.

2

Book Processing Pipeline

Using a prompt-based AI pipeline, educational books are processed and converted into structured JSON formats representing chapters and sub-concepts.

3

AI Question Generation

The AI analyzes structured data to automatically generate questions based on specific chapters or concepts, ensuring logical interconnection between related topics.

4

Dynamic Testing System

Tests are administered using Bloom’s Taxonomy, adjusting between Easy, Medium, and Difficult levels based on real-time student performance.

5

conceptual Gap Identification

The system identifies logical mistakes and conceptual misunderstandings, highlighting exactly where a student's fundamental knowledge is weak.

6

AI-Driven Analysis & Remedy

Final reports include concept-level mistake explanations, suggested improvements, and a curated remedy plan to bridge learning gaps effectively.

Key Features

📚

Concept-Wise Testing

Tests are structured around interconnected concepts, allowing students to verify their understanding layer by layer rather than just memorizing chapters.

🤖

AI Question Engine

Automated question generation using LLMs ensures fresh, relevant, and diverse testing material derived directly from structured book content.

📈

Bloom’s Taxonomy Analysis

Evaluates higher-order thinking skills by scaling questions across different cognitive levels to truly measure student mastery.

🛠️

Personalized Remedy Plans

Generates a custom "Remedy Plan" for every student, pinpointing weak sub-concepts and providing structured paths for improvement.
